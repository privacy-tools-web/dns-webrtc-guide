# DNS-over-HTTPS and WebRTC Leaks

A lightweight, static guide explaining DNS leaks and WebRTC IP leaks — what they are, how to detect them, and how to fix them. No dependencies, no tracking, no server — everything runs directly in your browser.

## Contents

- **Run the leak test first** — how to check whether your browser is currently leaking
- **Enable DNS-over-HTTPS** — step-by-step for Firefox, Chrome, Edge, and Safari
- **Fix WebRTC leaks** — how to disable or restrict WebRTC per browser
- **Verify your settings** — confirming the fixes are working
- **DoH provider comparison** — Cloudflare, Quad9, NextDNS, and others
- **FAQ & final checklist**

## Privacy

This is a fully static HTML page. No data is collected, transmitted, or stored. No external scripts, no analytics, no cookies of any kind.

## Usage

Download `dns-webrtc-guide.html` and open it in any modern browser. No build step, no install, no server required.

```bash
git clone https://github.com/privacy-tools-web/dns-webrtc-guide.git
open dns-webrtc-guide/dns-webrtc-guide.html
```
